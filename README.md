## Workflow Chosen
Summarizing meeting notes into structured action items.

## Who the User Is
The user is a project manager, business analyst, or team lead who needs to quickly turn rough meeting notes into a clear list of next steps.

## What Input the System Receives
The system receives unstructured meeting notes written in plain text. These notes may include incomplete sentences, rough ideas, task discussions, deadlines, or assigned owners.

## What Output the System Should Produce
The system should produce a structured summary of action items, including:
- task
- owner (if mentioned)
- deadline (if mentioned)
- priority or follow-up notes if relevant

## Why This Task Is Valuable Enough to Automate
This task is valuable because meeting notes are often messy and time-consuming to organize manually. A GenAI workflow can help save time, reduce missed follow-ups, and improve clarity after meetings. It is especially useful as a first-pass draft that a human can quickly review and finalize.
